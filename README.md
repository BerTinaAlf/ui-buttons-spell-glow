# UI Buttons Spell Glow

Foundry VTT UI mod that makes button borders transparent by default, then adds a blue-violet spell-like glow on hover, focus, active controls, and clicked states.

## Install Locally

1. Copy this folder into your Foundry user data modules directory:
   `Data/modules/ui-buttons-spell-glow`
2. Start or refresh Foundry VTT.
3. Enable **UI Buttons Spell Glow** in the world's module settings.

## Tuning

Edit `styles/spell-button-glow.css` and adjust the variables at the top if you want the magic effect to be brighter, softer, bluer, or more violet.
